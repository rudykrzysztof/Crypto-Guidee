## 🚀 Start

```bash
npm install
npm run dev
```

Aplikacja backendowa ruszy na porcie `3000`.

Frontend otwórz przez `frontend/index.html` albo podepnij później pod Vite / React.

## 📡 Endpointy

* `GET /api/prices`
* `GET /api/health`
